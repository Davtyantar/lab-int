Кинофраза

2016 год официально объявлен В.В.Путиным годом кино и профсоюзная организация решила поддержать инициативу Президента РФ, запустив интерактивный проект "Кинофраза".

Стать участником проекта сможет каждый. Для этого присылайте в течении всего 2016 года на электронный адрес VolynskayaTI @gaz.ru c пометкой "Кинофраза":

  • короткую известную фразу из кинофильма, которая давно уже стала для всех нас крылатой (в текстовом виде);
  • название фильма, в котором она прозвучала (в текстовом виде);
  • кадр из фильма, когда герои говорят эту фразу (картинка).

Ваша фраза будет выставлена на сайте в разделе "Проект "Кинофраза". Давайте вспоминать отечественые фильмы вместе!
